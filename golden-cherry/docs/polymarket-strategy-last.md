# Resolution Momentum Strategy (해결 직전 모멘텀 전략)

## 전략 개요

**핵심 아이디어**: 높은 확률 시장은 해결 직전에 해당 방향으로 급격히 수렴한다.

```
진입: 해결 24시간 이내 + 확률 75-92%
청산: 해결 4시간 전 또는 손절/익절
```

---

## 전략 근거

### 1. 시장 수렴 현상

| 해결까지 시간 | 시장 정확도 | 의미 |
|--------------|------------|------|
| 24시간 전 | 88.6% | 가격이 실제 확률과 괴리 |
| 12시간 전 | 90.2% | 수렴 시작 |
| 4시간 전 | 94.2% | 대부분 방향 확정 |

**출처**: Alex McCullough, Dune Analytics (2025년 3월)

### 2. 비대칭 수익 구조

```
75-92% 시장에서:

├── 확률: ~75-85% (캘리브레이션 고려)
└── 수익: +8~15%
```

### 3. 기대값 계산

```
파라미터:
- 진입가: 80%
- 실제 YES 확률: 76% (5%p 과대평가 가정)
- YES 시 수익: +10%
- 손절: -8%

기대값 = (0.76 × 10%) - (0.24 × 8%) = 5.68%
```

---

## 진입 조건

### 필수 조건 (모두 충족)

| 조건 | 값 | 이유 |
|------|-----|------|
| 확률 범위 | 75% ≤ P ≤ 92% | 너무 낮으면 불확실, 너무 높으면 수익 여지 없음 |
| 해결까지 시간 | 4시간 < T ≤ 24시간 | 수렴 구간 |
| 유동성 | ≥ $50,000 | 청산 가능성 확보 |


## 청산 조건

### 청산 우선순위

| 순위 | 조건 | 동작 |
|------|------|------|
| 1 | 손절: P&L ≤ -8% | 즉시 청산 |
| 2 | 익절: P&L ≥ +15% | 즉시 청산 |
| 3 | 트레일링: 최고점 대비 -5% | 수익 보호 |
| 4 | 시간: 해결 4시간 이내 | 예정 청산 |

### 손절 상세

```
기본 손절: -8%

동적 손절 (선택):
- 보유 4시간 후: -7%
- 보유 8시간 후: -6%
- 보유 12시간 후: -5%

이유: 시간이 지날수록 정보가 반영되므로 더 빠른 손절
```

### 트레일링 스탑

```
최고점 기록 후 5% 하락 시 청산

예시:
- 진입: 80%
- 최고점: 88% (기록)
- 트리거: 83.6% (88% × 0.95)
- 88% → 83.6% 하락 시 청산
```

## 기대 성과

### 시나리오별 분석

| 시나리오 | 확률 | 수익률 | 기여 |
|----------|------|--------|------|
| YES + 익절 | 20% | +15% | +3.0% |
| YES + 시간청산 | 50% | +8% | +4.0% |
| YES + 트레일링 | 8% | +5% | +0.4% |
| NO + 손절 | 22% | -8% | -1.8% |

**거래당 기대값: +5.6%**

### 월간/연간 추정

```
월 40회 거래 × 4% 기대값 × 2% 포지션 = 월 3.2%
연간 (복리): 약 46%
```

## 리스크 관리

### 최악의 시나리오

| 시나리오 | 손실 | 대응 |
|----------|------|------|
| 연속 NO 5회 | -8% | 일일 한도로 정지 |
| 손절 슬리피지 | -12% | 유동성 필터 강화 |
| 전체 시장 급변 | -10% | 노출 한도 50% |

### 필수 규칙

1. **손절 절대 엄수**: -8% 도달 시 무조건 청산
2. **분산 필수**: 최소 15개 시장 이상
3. **일일 한도**: -5% 도달 시 당일 거래 중단
